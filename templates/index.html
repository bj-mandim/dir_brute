Descubra diretórios e arquivos ocultos em aplicações web. Insira a URL alvo e inicie o reconhecimento de infraestrutura.

Como Funciona

Três passos para o reconhecimento

01

Defina o Alvo

Insira a URL do site que deseja auditar. A ferramenta valida o formato e prepara o escaneamento automaticamente.

02

Escaneamento Inteligente

O DirBrute varre centenas de caminhos usando uma wordlist otimizada para encontrar diretórios e arquivos ocultos.

03

Analise os Resultados

Receba um relatório com códigos de status HTTP, identificando páginas administrativas, APIs expostas e recursos sensíveis.

A Iniciativa

Segurança com inteligência

Reconhecimento de Infraestrutura

Projetada para facilitar a fase de reconhecimento em auditorias de segurança. Uma interface intuitiva que remove a barreira do terminal, permitindo análises rápidas de infraestrutura web sem sacrificar a profundidade técnica.

Offensive Security

Construída sob a ótica de Segurança Ofensiva, unindo a robustez do Python/Flask com uma interface moderna e acessível. Ideal para profissionais e estudantes que necessitam de agilidade em testes de penetração e reconhecimento web.

Aviso Legal: Esta ferramenta é fornecida exclusivamente para uso educacional e testes em sistemas que você possui autorização explícita para testar. O uso indevido contra sistemas sem permissão é ilegal e antiético.
